# 実験データ分析ガイド: 主観的速度再現性の評価

## 研究概要
遠隔操縦における映像伝送遅延補償のための多視点映像混合手法の主観的速度再現性評価に関する実験データ分析手法

## 実験設計の理解

### 実験1: 既存手法の評価
- **目的**: 時間的線形関数による混合手法の主観的速度再現性測定
- **手法**: 調整法（Method of Adjustment）
- **測定項目**: 主観的速度と実際の速度の比較

### 実験2: 改良手法の効果検証
- **目的**: 再設計された混合関数の速度再現性向上効果の実証
- **比較**: 既存手法 vs 改良手法

## 1. データ前処理

### 1.1 データクリーニング
```python
# 外れ値の検出と処理
- Z-score法 (|z| > 2.5)
- 四分位範囲法 (IQR)
- 参加者毎の反応時間による異常値除去
```

### 1.2 データ構造の確認
- 参加者ID、条件、試行回数、測定値の整合性
- 欠損データの確認と処理方針決定

## 2. 記述統計分析

### 2.1 基本統計量
各条件における以下の算出：
- 平均値、中央値、標準偏差
- 最大値、最小値、四分位数
- 歪度、尖度

### 2.2 視覚化
- ヒストグラム（分布の確認）
- 箱ひげ図（条件間比較）
- 散布図（実際速度 vs 主観速度）
- 時系列プロット（学習効果の確認）

## 3. 主観的速度再現性の評価指標

### 3.1 精度（Accuracy）
```
相対誤差 = (主観速度 - 実際速度) / 実際速度 × 100%
絶対誤差 = |主観速度 - 実際速度|
```

### 3.2 一貫性（Consistency）
```
変動係数 = 標準偏差 / 平均値
試行内標準偏差（参加者毎）
```

### 3.3 線形性（Linearity）
```
回帰直線の決定係数 R²
回帰直線の傾き（理想値: 1.0）
```

## 4. 統計的検定

### 4.1 正規性検定
- Shapiro-Wilk検定
- Kolmogorov-Smirnov検定
- 正規確率プロット

### 4.2 条件間比較
**パラメトリック検定（正規分布の場合）:**
- 対応のあるt検定（2条件比較）
- 反復測定ANOVA（3条件以上）
- 多重比較（Bonferroni, Tukey HSD）

**ノンパラメトリック検定（非正規分布の場合）:**
- Wilcoxon符号付順位検定
- Friedman検定
- Dunn's多重比較

### 4.3 回帰分析
```python
# 線形回帰モデル
主観速度 = β₀ + β₁ × 実際速度 + ε

# 混合効果モデル（参加者間変動を考慮）
主観速度 = β₀ + β₁ × 実際速度 + β₂ × 条件 + 参加者ランダム効果 + ε
```

## 5. 効果量の計算

### 5.1 Cohen's d
```
d = (平均値1 - 平均値2) / 統合標準偏差
```

### 5.2 相関係数
- Pearson相関係数（線形関係）
- Spearman順位相関係数（単調関係）

### 5.3 決定係数（R²）
回帰モデルの説明力評価
